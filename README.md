## gitの使い方

gitの使い方

```
// githubから最新版の状態をとってくる
git pull

// ステータス確認
git status

// ファイル編集などをする

// gitにファイルを追加する
git add . 

// 変更をコミットする
git commit -m"コミットメッセージ"

// 変更をpushする（githubに反映される）
git push origin master

// githubで確認する

// また最初から
```
